# 💱 Currency Converter

A modern and interactive Currency Converter web application built using HTML, CSS, and JavaScript.
This project allows users to convert currencies in real-time, view country flags, and explore country and currency details with a beautiful animated UI.

##  Features
*  Real-time currency conversion
*  Dynamic country flag updates
*  Exchange rate fetching using API
*  Displays currency full forms
*  Displays country names
*  Animated star background
*  Modern responsive UI
*  Automatic updates when currencies change

##  Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
* Currency Exchange API
* FlagsAPI
* Font Awesome Icons

##  How It Works
1. User enters an amount.
2. Selects source currency.
3. Selects target currency.
4. App fetches live exchange rate.
5. Converted amount is displayed instantly.
6. Flags, country names, and currency names update dynamically.
